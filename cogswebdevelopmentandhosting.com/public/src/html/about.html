About

Cogs Web Development and Hosting is a small business that specializes in a variety of services. Such as web design, web development, website hosting, web applications, and internet marketing.

History

Founded in 2017, it has steadily grown into the company that it is today, striving to bridge the gap between technologies of old with new.

Vision

The vision of Cogs Web Development and Hosting, is to strive to deliver websites and web applications to small businesses in the local community and abroad. To research and develop cutting edge tools to help improve the process and workflow of development across various platforms and code bases.

  • Bridge the gap between Web 2.0 and Web 3.0
  • Help open source communities and contribute to open source software.
  • To research and develop new tools and software

Goals

The goals of Cogs Web Development and Hosting for the time being is to create better open source tools, websites, and web applications in order to promote the company and drive traffic and profits.

  • To add more interactivity towards this site
  • Research into Web 3.0 technologies
  • Contribute to existing open source software
